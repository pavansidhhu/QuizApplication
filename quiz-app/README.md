# Quiz App

A full-stack Quiz Application built with Spring Boot, Angular, and MongoDB.

## Features
- **Admin**: Upload Quizzes (PDF), View Student Results.
- **Student**: Take Quizzes, View Scores.
- **PDF Parsing**: Automatically extracts questions from uploaded PDF files.

## Tech Stack
- **Backend**: Java, Spring Boot, Spring Data MongoDB, Apache PDFBox.
- **Frontend**: Angular, Vanilla CSS.
- **Database**: MongoDB.
